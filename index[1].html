    🧠 Powered by MediaPipe + OpenCV + scikit-learn + Claude AI

    How the ML analysis works

    The Python backend uses MediaPipe Face Mesh to detect 468 facial landmarks, then applies geometric algorithms to classify your face shape, OpenCV color analysis for skin undertone, and scikit-learn KMeans for dominant color extraction. Claude AI then generates personalized looks from those measurements.
